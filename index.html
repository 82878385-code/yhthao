城市环境服务权限配置培训教程

确保您的移动设备（Android/HarmonyOS 及 iOS）权限设置准确无误，保障“作业云”APP的定位、考勤、后台运行等核心功能稳定。

— 本手册适用于 Android、HarmonyOS 及 iOS 主流版本手机 —

跨品牌配置差异说明

由于各手机厂商（如华为、小米、OPPO、vivo、苹果等）的系统定制，设置界面的名称和位置可能有所不同。请根据教程中的功能描述和关键词灵活查找和配置，确保功能被正确开启。

步骤一： APP 内权限检查（授权管理）

（本节主要针对 Android/HarmonyOS 设备）

1

进入 APP 授权管理页面

按照以下路径操作：
打开城市环境服务 → 点击“作业检查” → 点击右上方“授权管理”

2

逐项开启所有系统权限 (显示为☑️)

在“授权管理”页面中，您需要确保以下所有权限的开关都处于开启状态，尤其是**后台位置信息**。

✓ 相机 ✓ 存储/文件访问 ✓ 位置信息 (前台) ✓ 后台位置信息 ✓ 运动与健身 ✓ 允许后台活动 ✓ 消息通知 ✓ 蓝牙 ✓ 关闭省电模式 ✓ 自启动

步骤二： 手机系统设置检查与优化

（本节主要针对 Android/HarmonyOS 设备）

3

状态栏基础开关检查 (网络与定位)

下滑手机屏幕顶部状态栏，确保以下四个核心开关全部处于开启状态：

📡 移动数据
🌐 WLAN/WiFi
📍 位置信息/定位
🅱️ 蓝牙
4

APP 自启动权限配置 (确保后台运行)

路径：手机设置 → 应用管理 (或 权限管理) → 找到 城市环境服务。

  • 若为“自动管理”模式：

    必须关闭“自动管理”，切换至手动管理，并开启以下三项开关：

    开机自启动、后台自启动、关联启动

  • 若为“自启动管理”模式：

    直接开启“城市环境服务”的自启动和关联启动开关。

5

电池/电量设置深度优化 (防止APP被系统“杀死”)

目标：所有与“作业云”APP相关的节能限制全部关闭或设置为无限制。

  • 省电模式： 立即关闭“省电模式”、“超级省电模式”以及“自动进入省电模式”。
  • 耗电策略/省电策略： 查找应用耗电策略，设置为 无限制 或 允许后台高耗电。
  • 耗电行为控制： 设置为 完全允许后台行为 (不限制应用的任何后台行为)。
  • 网络连接： 在“更多电池设置”中，务必开启“休眠时始终保持网络连接” 和 关闭“熄屏5分钟断开网络连接”的选项。
  • 待机优化： 务必关闭“睡眠模式”或“睡眠智能待机优化”。
  • 电池优化： 确保设置为 不优化 或 关闭电池优化。

步骤三： 其他特殊系统设置检查

（本节主要针对 Android/HarmonyOS 设备）

📶 禁用省流量模式

在 手机管家 中找到 流量管理，务必关闭“省流量模式”开关。

后果： 开启此模式会阻止后台应用使用移动数据，导致考勤数据和消息无法实时传输。

🗑️ 禁用闲置权限移除

在 应用管理 → 城市环境服务 页面中，务必关闭以下功能：

  • “移除长期未使用的应用权限”
  • “暂停闲置应用的活动” / “应用速冻”

🏃 运动与WLAN权限确认

在 应用信息 → 权限 中，再次确认以下权限已明确允许：

  • 运动与健身 / 身体活动信息
  • WLAN (本地网络)

步骤四： iOS (苹果) 设备权限配置与优化

APP 名称提醒

请在 App Store 下载名为 “城市环境服务” 的应用。以下步骤均基于此应用。

1

初始授权弹窗配置（首次打开）

  • 无线数据/网络： 选择 无线局域网与蜂窝网络。
  • 隐私政策： 选择 同意并继续。
  • 运动与健康/健身： 选择 好（允许访问活动与体能训练记录）。
  • 初始定位： 定位权限弹窗中选择 使用App时允许。
2

定位权限深度配置 (保障后台考勤)

这是最关键的步骤，必须手动进入设置修改，确保后台定位功能正常工作：

  • 进入设置： 路径为 设置 → 找到 城市环境服务 (或 设置 → App → 城市环境服务)。
  • 位置权限： 点击 位置，将权限从“使用App期间”改为 始终。
  • 精确位置： 务必 开启 精确位置 开关。
3

后台与其他关键设置

在 设置 → 城市环境服务 页面内，确保以下权限已明确开启：

✓ 后台 App 刷新 ✓ 通知 ✓ 无线数据 ✓ 运动与健身
